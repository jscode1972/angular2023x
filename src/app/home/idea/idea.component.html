資料夾架構最佳實踐!!!! (demo就用這個) 
Folder for each Angular Module
- Root Module (根目錄)
- Feature Modules (功能模塊)
  - components, directives, pipes, 
  - pages (元件.component.ts -> 元件.page.ts)
      每個路由都是一個頁面。路由可能包含多個元件。
- Shared Module (共享元件可以放在獨立的元件資料夾)
    components  (models / guards / mocks)
-  Core Module
