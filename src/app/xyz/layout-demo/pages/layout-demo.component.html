CSS Box Model 區塊模型 
      每一個標籤都佔一個方形區塊(block) 想像一堆框框
        行內模式 inline   (a, span, img) 
        區塊模式 block 自動換行 (h1, div, ul, table..)
        隱藏模式 none 額外透過 css display 設定
      區塊填塞 padding 內容和邊框距離 (4個方向)
      區塊邊框 border
      外邊距 (很重要) 相鄰區塊間的距離 (4個方向) => 通常講的是兄弟區塊,不是內外
      背景色 (區塊背景)
      區塊寬/高 (限區塊模式block, 即span不可設定)
    
    FlexBox 
      多層次結構,每次只看相對的內外層關係 (遞迴處理)
      外層 display:block (內層一律佔滿跳行)
      外層 display:flex; 搭配 justify-content & align-items 成對使用 
        內層 flex: none;
CSS Flex 屬性一點也不難 (超清楚)
